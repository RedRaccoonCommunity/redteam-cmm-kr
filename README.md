# Red Team Capability Maturity Model - 한글 번역 

[Red Team CMM](https://www.redteammaturity.com/) 프로젝트의 한글 번역판입니다. 

모든 크레딧은 원본 프로젝트의 원작자와 기여자들에게 있습니다 - [링크](https://www.redteammaturity.com/about). 

한글 번역판의 번역 및 관리는 [레드라쿤](https://www.redraccoon.kr/)에서 담당하고 있습니다. 

## 구조 

본 프로젝트는 [cmm.redraccoon.kr](https://cmm.redraccoon.kr)에서 호스팅되고 있습니다. 

인하우스 레드팀들의 CMM 평가 및 점검을 용이하게 하기 위해 `RedTeam-CMM-V1-한글-배포.xlsx`과 `RedTeam-CMM-V1-한글-배포.csv`를 배포하고 있습니다.
